# Sign Language Recognition System

Real-time sign language recognition using MediaPipe and TensorFlow.

## Features
- Detects hand landmarks using MediaPipe
- Recognizes ASL letters A-Z
- Real-time word formation
- 95%+ accuracy with data augmentation

## Requirements
pip install opencv-python mediapipe tensorflow numpy scikit-learn matplotlib


## Usage
1. Run cells 1-2 to collect training data
2. Run cells 3-6 to train the model
3. Run cell 8 for real-time recognition
